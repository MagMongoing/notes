### 列表
列表切片和索引

- extend
```
a = [1,2,3]
b = [2, 3]
a.extend(b)
a
[1, 2, 3, 2, 3]

```
- insert
```
a.insert(2,'s')
a
[1, 2, 's', 3, 2, 3]
```
- pop & remove
```
a.pop()  # 默认一处最后一个，可以传入移除的位置
3
a
[1, 2, 's', 3, 2]

a.remove('s')
a
[1, 2, 3, 2]
```
